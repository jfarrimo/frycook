 35      The Cookbook class is the base class for all cookbooks to be created from.

 38   
 39      In your subclass you will fill the class-level variable recipe_list with

 41      The recipes will be applied in the order they're defined in the list.

 43      Example:: 
 44   
 45        from frycook import Cookbook

 48        class MyOwnCookbook(Cookbook):
